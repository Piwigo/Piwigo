Siti in remoto

Piwigo offre la possibilità di usare diversi server per immagazzinare le immagini della vostra galleria. Ciò può essere utile se la vostra galleria è installata su uno spazio limitato e avete una grande quantità di immagini da mostrare.

  1. modificare il file tools/create_listing_file_local.inc.php agendo sui parametri come $conf['prefix_thumbnail'] o $conf['use_exif'].
  2. via FTP, caricare i file tools/create_listing_file.php e tools/create_listing_file_local.inc.php (modificato) sul Vostro sito remoto, nella stessa directory con gli album che compongono la vostra galleria (come la directory galleries di questo sito). Per esempio, supponiamo che possiate accedere a http://esempio.it/galleries/create_listing_file.php.
  3. andate su Amministrazione » Tools » Gestione dei siti. Richiedete la creazione di un nuovo sito, per esempio http://esempio.it/galleries.

  4. Un nuovo sito remoto è registrato. 4 azioni sono ora possibili :

    1. generare la lista : lancia una richiesta remota per generare il file di listing remoto
    2. sincronizzare : legge il file remoto listing.xml e sincronizza con il database locale.
    3. pulire : cancella il file di listing remoto.
    4. cancellare : cancella il sito remoto (e tutti gli elementi associati) dal database.
